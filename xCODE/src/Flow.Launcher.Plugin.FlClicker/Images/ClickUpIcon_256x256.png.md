# The Icon file


![ClickUpIcon 256x256](ClickUpIcon_256x256.png)

This Icon-Image is used as the preliminary icon when FlClicker-stuff displays stuff in the FlowLauncher.exe. 

Its size is 256x256 pixel and can be virtually anything. 

Its format is recommended to be a *.png-file. 

To load it correctly some configurations have to be made with care

## plugin.json
Within the [plugin.json](../plugin.json.md)-file this icon has to be referenced as follows: 

    "IcoPath": "Images\\ClickUpIcon_256x256.png",
